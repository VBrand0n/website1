En este sitio web encontraremos información sobre el proyecto de aula del estudiante Brandon Arias en el módulo de producción multimedial, he hecho esto con ayuda de la información que nos brindó el profesor en clases y con mucha ayuda de el auto-prendizaje. En este proyecto usé varias herramientas como Adobe Spark,Animaker y Visual Studio. Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código
